# 🍔 BurgerHub — Restaurant E-Commerce Website

> A fully designed, multi-page restaurant e-commerce website built for a client who needed a premium online presence for their burger restaurant.

![BurgerHub Homepage](assets/screenshots/homepage.jpg)

---

## 📌 Project Overview

**BurgerHub** is a complete restaurant website design and frontend project featuring a dark, modern aesthetic with bold red accents. The client requested an e-commerce website for their restaurant — this project delivers a full multi-page experience covering everything from online ordering to brand storytelling.

**Client Brief:** _"Make me an e-commerce website for my restaurant."_  
**Deliverable:** A fully designed, multi-page restaurant website with ordering functionality, menu browsing, brand story, and contact/location pages.

---

## 🎯 Project Goals

- Build a visually striking, dark-themed restaurant website
- Allow customers to browse the menu and customize orders online
- Showcase the restaurant's brand story and community
- Provide easy access to contact info, locations, and opening hours
- Include a newsletter signup and app download section

## 🎨 Design System

| Element | Value |
|---------|-------|
| **Primary Color** | `#E8341C` (Red) |
| **Background** | `#0A0A0A` (Near Black) |
| **Card Background** | `#1A1A1A` |
| **Text Primary** | `#FFFFFF` |
| **Text Secondary** | `#9CA3AF` |
| **Accent / Highlight** | `#F97316` (Orange-Red) |

**Typography:**
- Display: Bold sans-serif (large weight headlines)
- Body: Clean sans-serif for readability on dark backgrounds

**Design Style:** Dark mode · Bold typography · Card-based layout · Red CTA buttons · Subtle gradients

## 🚀 Getting Started

### Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/burgerhub.git
cd burgerhub
```

### Open Locally
```bash
# Simply open in browser
open index.html

# Or use Live Server (VS Code extension)
# Right-click index.html → Open with Live Server
```

### Deploy to GitHub Pages
```bash
git add .
git commit -m "Initial commit - BurgerHub website"
git push origin main
# Then enable GitHub Pages in repo Settings → Pages → Deploy from main branch
```
